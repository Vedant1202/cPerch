macOS 14+ Fully local · no network Zero permissions

Everything your agents are doing, in one glance

cPerch turns "which window was that agent in again?" into a quick look at the menu bar.

Every session, unified

Terminal and Claude desktop sessions listed together — needs-you first — across all your projects.

One-click Jump

Raises and focuses the existing window — the exact terminal tab or the desktop app. Never a duplicate.

Calm notifications

Opt in by kind — needs-input, error, completion. Focus/DND aware, with tap-to-open. Quiet by default.

Accessibility-first

Shape-coded status (not color alone), a high-contrast mode, VoiceOver labels, and reduce-motion support.

Private by design

Reads only your local ~/.claude directory. No network, ever. Transcript content never leaves your Mac.

Hotkey & launch at login

Toggle the roster from anywhere with ⌘⌥`, and have cPerch start with your Mac — both optional.

Zero permissions

No Accessibility permission, no Input Monitoring, no auth tokens. Detection works straight from the filesystem.

Read-only

cPerch reads ~/.claude to detect sessions. It never writes to it or changes anything.

No network

Fully offline. cPerch makes no network requests and never transmits transcript content.

Get started

No packaged release yet — build from source. Command Line Tools only, no full Xcode required.

# clone, build, and launch
git clone https://github.com/Vedant1202/cPerch.git
cd cPerch
./build.sh            # → dist/CPerch.app (ad-hoc signed)
open dist/CPerch.app  # menu-bar app — look for the dot, top-right
